O TecnosA é um aplicativo web de gestão técnica desenvolvido com Ionic + Angular, integrado ao Firebase (Firestore), voltado para o controle de clientes, equipamentos e ordens de serviço (OS).
A aplicação permite cadastrar, visualizar, editar e excluir dados em tempo real, funcionando como um sistema básico de gerenciamento para assistências técnicas, manutenção de equipamentos e suporte técnico.
